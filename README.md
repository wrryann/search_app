### how to run locally 

**attetion: you must have rust and cargo installed!!**

1. clone repository using git clone
```bash
git clone https://github.com/wrryann/search_app.git
```

2. go to directory

```bash
cd search_app 
```

3. run it using cargo

```bash
cargo run 
```

### used crate:

[pokemon-rs](https://crates.io/crates/pokemon-rs)
